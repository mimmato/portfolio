Welcome to my API!

This API has been created as part of my GitHub portfolio for mimmato/portfolio. It provides endpoints to manage user data for testing purposes.

Feel free to explore the available data and browse through the various endpoints to see how the API works.

Available Endpoints

GET /users

Retrieve all users from the database.

Example: GET http://localhost:3000/users

GET /users/:id

Retrieve a specific user by ID from the database.

Example: GET http://localhost:3000/users/123

POST /users

Create a new user and store it in the database.

Example: 

POST http://localhost:3000/users
Content-Type: application/json

{
    "name": "John Doe",
    "email": "john.doe@example.com"
}

DELETE /users/:id

Delete a user by ID from the database.

Example: DELETE http://localhost:3000/users/123

PATCH /users/:id

Update an existing user in the database.

Example: 

PATCH http://localhost:3000/users/123
Content-Type: application/json

{
    "name": "Updated Name",
    "email": "updated.email@example.com"
}
